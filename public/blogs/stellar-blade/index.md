# 游戏介绍

《剑星》（Stellar Blade）是一款由韩国 Shift Up 开发的大型**第三人称动作角色扮演游戏（ARPG）**，融合了动作冒险与科幻元素。玩家扮演拥有强大战斗能力的女战士夏娃（Eve），在充满敌意的外星环境中展开冒险。

![剑星游戏截图](https://raw.githubusercontent.com/zzblove/zzblove-imgbed/main/blogs/stellar-blade/screenshot.webp)

## 游戏特色

- **精美画面与角色设计** — 由知名画师金亨泰亲自操刀，角色形象极具辨识度
- **流畅战斗系统** — 无缝切换武器和技能，攻击与闪避衔接自然爽快
- **丰富剧情与世界观** — 未来科幻世界，人类与外星生物战争持续
- **探索解谜元素** — 大型场景探索、支线任务与挑战
- **动态环境互动** — 场景会根据玩家行动发生变化
